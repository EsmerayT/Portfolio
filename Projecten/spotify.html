Dit project is een Spotify Search App. Een React App, die de Spotify API gebruikt om data te fetchen. De appliciatie start met een login scherm, om de app te machtigen je Spotify-accountgegevens te gebruiken. Vervolgens wordt je doorverwezen naar de Applicatie. Door middel van de zoekbalk kan worden gezocht naar artiesten, albums en afspeelijsten. Wanneer je op één van de resultaten klikt zal je doorverwezen worden naar Spotify!
